## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

```
Git
VS Code
Node
```

### Setup

```
cd my-folder
gh repo clone cosmosha/React-Todo-APP
cd React-Todo-APP
```

### Install

```
npm install
```

### Usage

```
npm run build
npm start
```

### Run tests

```
npm test
```
